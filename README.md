# Projeto de Software — Resenhas

Repositório de resenhas de artigos da disciplina **Projeto de Software** (Engenharia de Software · PUC Minas).

---

## Como estão organizadas

Cada resenha fica em `resenhas/` como um arquivo `.md`.

Cada arquivo segue o formato pedido pelo professor:

1. **Conceito** — ideia central do artigo
2. **Análise** — interpretação e reflexão crítica
3. **Aplicação prática** — como isso se aplica no mundo real
